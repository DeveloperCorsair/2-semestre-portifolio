# Portfólio — Henrique Moreira

Portfólio pessoal desenvolvido como projeto do 2º semestre da Fatec Jessen Vidal.

## 🛠️ Tecnologias

- HTML, CSS e JavaScript puro
- Google Fonts (Syne, DM Sans, JetBrains Mono)

## 📁 Estrutura

```
├── index.html
├── css/
├── ├── resp.css
├── ├── style.css
├── script.js
└── img/
```

## ⚙️ JavaScript

| Recurso | Descrição |
|---|---|
| `if/else` | Saudação dinâmica no header conforme o horário |
| `for` | Renderização de projetos e cursos a partir de arrays |
| `function` | `renderizarProjetos()`, `renderizarCursos()`, `exibirSaudacao()` |

## 🚀 Como rodar

Abra o `index.html` direto no navegador. Sem dependências, sem build.
